# karabiner

Karabiner-Elements rules (macOS).

## Files

- `alacritty.json` — rules that only apply while Alacritty is focused.

## What it does

- Left Control + Return → maximize the Alacritty window (windowed, not fullscreen).
- Swaps Left Command and Left Control inside Alacritty.

## Notes

macOS only. The maximize rule runs `$HOME/.dotfiles/alacritty/scripts/maximize-windowed`.
