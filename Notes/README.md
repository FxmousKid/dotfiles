# Notes

Personal reference notes and a workspace bootstrap script.

## Files

- `desktop.sh` — clones my project repos (42 school, LeetCode, Advent of Code,
  university) into a fresh workspace. Re-running pulls the existing ones (`--ff-only`).
- `tools.md` — a running list of useful tools and links.

## Notes

This is personal. Read `desktop.sh` before running it on a new machine:

```sh
./Notes/desktop.sh
```
