# hyprland

Hyprland config (Wayland compositor, Linux).

## Files

- `hyprland.conf` — mostly stock, with my programs, autostart, env vars,
  gestures, input, and keybinds.

## Highlights

- Terminal: kitty · file manager: dolphin · menu: `wofi --show drun`
- Layout: dwindle
- Autostart: terminal, network/Bluetooth applets, Waybar, Hyprpaper, Firefox.

## Notes

Linux + Wayland only.
