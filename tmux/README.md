# tmux

Tmux config with plugins and session restore.

## Files

- `tmux.conf` — plugins, mouse mode, status bar, pane navigation and borders,
  resurrect, and terminal passthrough for image previews.

## Plugins (via TPM)

- tpm · tmux-sensible · tmux-continuum · tmux-resurrect · feqzz/tmux-mpv-info

TPM installs itself on first run if it's missing.
