# halloy

Config for the Halloy IRC client.

## Files

- `config.toml` — anonymized template (server, nick, channels). Fill in your own
  values.

## Notes

Not linked by the installer — copy it into place yourself if you want it. Keep
real passwords out of the repo (set them on the machine or use your keychain).
`dangerously_accept_invalid_certs` is left off; only enable it for a server with
a self-signed cert.
