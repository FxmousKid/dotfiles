# alacritty / scripts

macOS helper that maximizes Alacritty without going fullscreen. Karabiner runs it.

## Files

- `maximize-windowed` — the shell entry point Karabiner calls.
- `maximize-windowed.swift` — the Swift source.
- `maximize-windowed.bin` — the compiled binary.

## Notes

macOS only. Karabiner runs `$HOME/.dotfiles/alacritty/scripts/maximize-windowed`.
